# Userscripts
Personal collection of tampermonkey userscripts

### Cmoa free contents filter
Filter genres (BL, TL, light novels) from [https://www.cmoa.jp/freecontents/]() and allow individual series to be blacklisted with an X button on the cover image


### Hidive no subs
Hides subtitles on hidive anime


### Cmoa free contents filter
Use vertical text on aozora bunko books
